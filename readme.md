# SourcePawn VSCode Remote Debugger [WIP]

This is a debug adapter for SourcePawn language which allows for remote debugging of scripts. It requires specific setup on
application side. (https://github.com/Garey27/sm-debugger)

## Build and Run

* Clone the project
* Open the project folder in VS Code.
* Press `F5` to build and launch SourcePawn Debuger in another VS Code window. In that window:
  * Open workspace with amxmodx source files.
  * Switch to the debug viewlet and press the gear dropdown.
  * Select the debug environment "sourcepawn Debugger".
  * Press `F5` to start debugging.
